## 問題

リンクリストの先頭 head が与えられたとき、リンクリストにサイクルがあるかどうかを判定します。リスト内に、次のポインタを連続的に追跡することで再度訪問できるノードがある場合、リンクリストにサイクルがあります。リンクリストにサイクルがある場合は true を返します。そうでない場合は false を返します。
