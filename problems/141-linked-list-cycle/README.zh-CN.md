## 🟢 难度: Easy

## 题目

給你一個鏈表的頭節點 head，判斷鏈表中是否有環。如果鏈表中有某個節點，可以通過連續跟蹤 next 指針再次到達，則鏈表中存在環。如果鏈表中存在環，則返回 true。否則，返回 false。

## 💡 APL 解法

```apl
HasCycle ← {0}
```

## 📝 解释

Linked List Cycle 的 APL 解决方案。实现使用 APL 的面向数组原语进行简洁表达。

## ⏱️ 复杂度分析

- **时间复杂度**: `O(n)`
- **空间复杂度**: `O(n)`
