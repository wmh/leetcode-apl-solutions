## 🟢 難度: Easy

## 題目

給你兩棵二元樹的根節點 p 和 q，編寫一個函數來檢驗這兩棵樹是否相同。如果兩個樹在結構上相同，並且節點具有相同的值，則認為它們是相同的。

## 💡 APL 解法

```apl
IsSameTree ← {⍺≡⍵}
```

## 📝 解釋

Same Tree 的 APL 解決方案。實現使用 APL 的面向陣列原語進行簡潔表達。

## ⏱️ 複雜度分析

- **時間複雜度**: `O(n)`
- **空間複雜度**: `O(1)`
