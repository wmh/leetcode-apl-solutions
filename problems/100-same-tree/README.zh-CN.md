## 🟢 难度: Easy

## 题目

給你兩棵二叉樹的根節點 p 和 q，編寫一個函數來檢驗這兩棵樹是否相同。如果兩個樹在結構上相同，並且節點具有相同的值，則認為它們是相同的。

## 💡 APL 解法

```apl
IsSameTree ← {⍺≡⍵}
```

## 📝 解释

Same Tree 的 APL 解决方案。实现使用 APL 的面向数组原语进行简洁表达。

## ⏱️ 复杂度分析

- **时间复杂度**: `O(n)`
- **空间复杂度**: `O(1)`
